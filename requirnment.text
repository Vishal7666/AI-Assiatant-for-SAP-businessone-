# ── AI / LLM APIs ─────────────────────────────────────────────────────────────
google-genai                # Gemini API (general chat, web search grounding)
groq                        # Groq API (NL→SQL for SAP B1 database queries)

# ── Web Server / API ──────────────────────────────────────────────────────────
fastapi                     # REST API framework
uvicorn                     # ASGI server to run FastAPI
pydantic                    # Request/response data validation (bundled with FastAPI)

# ── Database ──────────────────────────────────────────────────────────────────
pyodbc                      # MS SQL Server connection via ODBC Driver 17
